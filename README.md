# SOC Toolkit

Uma coleção de ferramentas, plataformas e recursos úteis para profissionais de Cybersecurity, especialmente para atuação em Blue Team, SOC, DFIR, Threat Hunting e Incident Response.

O objetivo deste projeto é centralizar recursos utilizados no dia a dia e facilitar o acesso a ferramentas de análise, investigação e defesa.

## 🤝 Como contribuir

Conhece alguma ferramenta que deveria estar nesta lista?

Abra uma Issue ou envie um Pull Request.

Toda contribuição é bem-vinda.

# Threat Intelligence

| Ferramenta | Descrição | URL |
|------------|-----------|-----|
| VirusTotal | Plataforma de Threat Intelligence para consulta de hashes, IPs, domínios, URLs e arquivos suspeitos utilizando diversos mecanismos de detecção. | https://www.virustotal.com |
| AbuseIPDB | Consulta a reputação de endereços IP baseada em denúncias da comunidade. Muito útil para validação de indicadores de comprometimento (IoCs). | https://www.abuseipdb.com |
| AlienVault OTX | Plataforma colaborativa de Threat Intelligence com indicadores de comprometimento e pesquisas sobre ameaças. | https://otx.alienvault.com |
| URLScan.io | Analisa URLs, captura screenshots, identifica tecnologias utilizadas e registra informações sobre páginas web. | https://urlscan.io |
| GreyNoise | Ajuda a diferenciar scanners da Internet de atividades potencialmente maliciosas. | https://www.greynoise.io |
